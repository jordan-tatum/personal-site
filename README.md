## 📋 About

This is my personal portfolio website showcasing my journey as a self-taught Data Engineer. It includes a homepage, about me, projects, and a resume page — all built from scratch using Bootstrap 5 with a custom navy and grey design system.

- HTML5
- CSS3
- [Bootstrap 5.3](https://getbootstrap.com/)
- [Bootstrap Icons](https://icons.getbootstrap.com/)
